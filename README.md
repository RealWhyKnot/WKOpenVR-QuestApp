# WKOpenVR Quest App

Quest App is the WKOpenVR module for Quest companion setup, APK install, and Quest-side support tooling used by WKOpenVR features.

Download the latest `WKOpenVR-QuestApp-v...-Setup.exe` from Releases and run it. The installer installs WKOpenVR, registers the SteamVR driver, and enables Quest App. Other WKOpenVR modules can be enabled from the Modules tab or by running another module installer.
